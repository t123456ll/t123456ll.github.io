# Introduction介绍:
目测会用来分享一下最近喜欢的电影🎬，书📖，音乐🎵什么的。

电影🎬 Movies：

最近有点想看安眠医生。身为恐怖片的狂热粉丝，我早已把闪灵无论是从小说原著，还是电影，或者是影评都看了好几遍，每一次都让我由衷感慨，斯蒂芬金是优秀的恐怖小说作家，电影也是经典恐怖片杰作。仍然记得初中时的自己，拿着一部当时的智能机（按键➕劣质触摸屏的那种），躲在宿舍的被窝里，偷偷看恐怖小说的日子。当时除了闪灵，还有德州电锯杀人狂，想想当时自己还真是硬核😂。

虽然觉得后续超越前作的可能性并不大，但是还是很想去看看，然后回忆一下过去看恐怖片小说的日子。
